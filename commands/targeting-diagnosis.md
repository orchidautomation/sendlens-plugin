# Targeting Diagnosis

Use the `targeting-diagnosis` skill when the user wants to decide who to target next, whether the current audience is wrong, how to redefine the audience, or which GTM plays to test from observed campaign evidence.

Arguments: $ARGUMENTS

If arguments are provided, treat campaign name or Instantly tag as the preferred analysis scope before synthesizing the targeting recommendation.

Use any business context in the arguments as context, not as measured evidence. Keep observed campaign evidence, sampled lead signals, and business assumptions clearly separated.
