# Support and maintenance

SendLens is intended to be maintained as provider APIs, AI hosts, and MCP conventions change.

## Support areas {#support-areas}

- Installation and host setup.
- Provider configuration and key rotation guidance.
- Evidence interpretation and workflow adoption.
- Single-tenant HTTP or container deployment review for enterprise users.
- Provider API drift triage for Instantly and Smartlead read-only surfaces.

## Boundaries {#boundaries}

Support does not require users to share raw customer campaign data in public channels. Use redacted examples, synthetic data, or private approved support paths.
