## Release story {#release-story}

SendLens has evolved from an Instantly-focused local analytics plugin into a provider-normalized, evidence-aware outbound intelligence layer for AI hosts. The core story is:

1. Local-first Instantly analytics.
2. Stronger MCP response contracts and evidence language.
3. Specialist workflows for setup, analysis, strategy, copy, and launch operations.
4. Smartlead V1 read-only parity work.
5. Enterprise-ready transport and deployment options.
6. Public docs and commercial readiness for closed-source distribution.
