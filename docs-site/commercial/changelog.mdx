1. Local-first outbound analytics for provider data.
2. Instantly campaign, reply, copy, sender, and lead analysis.
3. Smartlead V1 read-only provider parity, with support-gated Smart Delivery caveats.
4. All-provider normalization for cross-provider comparison.
5. Stronger MCP response contracts, evidence language, and specialist workflows.
6. Enterprise-ready transport, public docs, and commercial readiness.
