# Demo mode

Demo mode lets you try SendLens without connecting a provider account. It uses synthetic campaigns, replies, leads, and copy examples so you can inspect the workflow without exposing customer data.

## When to use demo mode {#when-to-use}

- Evaluating SendLens before provider key approval.
- Training a teammate on evidence labels.
- Testing host integration after installation.
- Demonstrating a workflow without private campaign data.

## What demo mode does not prove {#limits}

Demo mode does not prove provider credentials, live refresh performance, or the completeness of your real workspace. Use it for workflow orientation, then configure Instantly or Smartlead for production analysis.
