# Configuration reference

## Provider variables {#provider-vars}

- `SENDLENS_INSTANTLY_API_KEY`: Instantly read-only API key.
- `SENDLENS_SMARTLEAD_API_KEY`: Smartlead read-only API key.
- `SENDLENS_PROVIDER`: `instantly`, `smartlead`, or `all`.

## Cache variables {#cache-vars}

- `SENDLENS_CLIENT`: client or workspace identity for cache separation.
- `SENDLENS_DB_PATH`: explicit local DuckDB path.
- `SENDLENS_STATE_DIR`: explicit local state directory.

## Secret handling {#secret-handling}

Do not paste secret values into chat or docs. Store them in your host environment or secret manager and restart the host after changing them.
