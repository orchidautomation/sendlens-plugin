# Glossary

## Evidence class {#evidence-class}

A label that describes how strongly the available data supports an answer. See [Evidence model](/concepts/evidence-model).

## Local cache {#local-cache}

The DuckDB-backed store SendLens uses on the machine running the host or MCP server.

## Provider scope {#provider-scope}

The provider, workspace, campaign, or client boundary covered by an answer.

## System of action {#system-of-action}

A tool that can change provider state. SendLens is not a system of action.
