# `sendlens-campaign-strategist`

Turns validated SendLens findings into a campaign blueprint: audience, exclusions, problem, offer, angle, proof boundary, CTA, sequence architecture, personalization requirements, and experiment hypothesis.

Use it directly for focused “what campaign should we run?” requests. It does not draft full email bodies or make launch-ready claims. When the request is broad, `sendlens-analyst` invokes it automatically after diagnosis.
